cljamClojure for JavaScript

A Clojure interpreter that embeds in any JS/TS project. Run as a CLI, connect from Calva, or give LLMs a persistent REPL via MCP.

Get Started
Try it in the browser
View on GitHub
🧩

Embeddable

createSession() gives you a sandboxed Clojure runtime in any JS/TS project. Inject host capabilities, load libraries, evaluate strings.

🔌

nREPL compatible

Full TCP nREPL server. Calva, CIDER, and Cursive connect out of the box. Pair-program with an LLM in the same live runtime.

🤖

LLM-native

cljam-mcp is an MCP server giving Claude and other LLMs a persistent, stateful Clojure environment as tool calls.

📦

Library ecosystem

cljam-schema (malli-inspired validation), cljam-date, cljam-integrant — all installable from npm. Build your own libraries with cljam gen-lib-source.
